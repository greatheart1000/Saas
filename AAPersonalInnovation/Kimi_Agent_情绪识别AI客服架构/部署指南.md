# AI客服系统 - 依赖安装说明

## 安装步骤

### 方式1：使用requirements.txt（推荐）

```bash
cd backend
pip install -r requirements.txt
```

### 方式2：单独安装核心依赖

```bash
pip install fastapi uvicorn sqlalchemy pymysql
pip install pydantic pydantic-settings redis
pip install langchain langchain-openai openai
pip install chromadb rank-bm25
pip install loguru httpx python-dotenv
```

### 方式3：使用国内镜像源（更快）

```bash
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple fastapi uvicorn sqlalchemy pymysql
pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pydantic redis
```

## 数据库初始化

安装依赖后，运行：

```bash
python backend/init_db.py
```

## 启动服务

```bash
cd backend
python -m app.api.main
```

## 访问前端

- 对话监控: http://localhost:8000/static/conversations.html
- 知识库管理: http://localhost:8000/static/knowledge.html
- API文档: http://localhost:8000/docs
