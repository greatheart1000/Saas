# AI客服系统 - 功能增强说明

## 🎉 三个核心增强功能已实现！

### 1. 🪟 上下文窗口管理

**功能说明：**
- 为不同任务配置不同的上下文窗口大小
- 避免token浪费，提高响应速度

**配置参数：**
```env
CONTEXT_SENTIMENT_LIMIT=5         # 情绪分析：最近5条消息
CONTEXT_INTENT_LIMIT=5            # 意图识别：最近5条消息
CONTEXT_RESPONSE_LIMIT=10         # 生成回复：最近10条消息
CONTEXT_SUMMARY_LIMIT=20          # 生成总结：最近20条消息
```

**使用方式：**
```python
# 使用默认配置
response = await client.send_message(
    message="我想退款",
    session_id="user_001"
)

# 自定义上下文窗口
response = await client.send_message(
    message="我想退款",
    session_id="user_001",
    context_limit=20  # 这次使用最近20条消息
)
```

### 2. 📝 自动摘要功能

**功能说明：**
系统会自动在以下情况触发对话摘要：

#### 触发条件
1. **消息数达到阈值**（默认15条）
2. **时间间隔达到阈值**（默认10分钟）

#### 摘要执行的操作
1. 使用LLM生成摘要（100字以内）
2. 提取关键信息（订单号、手机号等）
3. 分析用户意图和情绪趋势
4. 保存到长期记忆（7天）
5. 压缩短期记忆（只保留最近N条）

**配置参数：**
```env
SUMMARY_TRIGGER_MESSAGE_COUNT=15  # 消息数阈值
SUMMARY_TRIGGER_TIME_MINUTES=10   # 时间间隔阈值
SUMMARY_AUTO_SUMMARIZE=true       # 启用自动摘要
SUMMARY_PRESERVE_KEY_MESSAGES=3   # 摘要后保留最近3条消息
```

**工作流程：**
```
用户发送消息
  ↓
更新短期记忆
  ↓
检查是否需要摘要：
  ├─ 消息数 >= 15？ ✅
  ├─ 时间间隔 >= 10分钟？ ✅
  └─ 从未摘要过且消息数 >= 5？ ✅
  ↓
触发自动摘要
  ├─ LLM生成摘要
  ├─ 提取关键事实
  ├─ 分析意图和情绪
  ↓
保存到长期记忆（7天）
  ↓
压缩短期记忆
```

### 3. 🎭 Persona拟人化系统

**功能说明：**
提供5种不同的客服人格，让AI回复更贴近真实人类

#### 可用Persona类型

**1. Friendly - 亲切友好型（默认）**
- 特点：温和友善，充满同理心
- 使用少量emoji：😊 🌟 💖
- 示例："您好呀！我很乐意帮您处理这个问题~ 😊"

**2. Cute - 可爱热情型**
- 特点：活泼可爱，充满活力
- 使用较多emoji：✨ 💪 🎉
- 示例："好哒~ 包在我身上！一定帮您搞定！✨"

**3. Mature - 成熟冷静型**
- 特点：稳重专业，值得信赖
- 不使用emoji
- 示例："根据您描述的情况，建议您采取以下步骤..."

**4. Professional - 专业严谨型**
- 特点：严谨专业，高效准确
- 不使用emoji
- 示例："针对您的问题，解决方案如下：1. ... 2. ..."

**5. Humorous - 幽默风趣型**
- 特点：风趣幽默，轻松愉快
- 使用适量emoji：😄 🎭 💡
- 示例："这个问题就像打地鼠一样，我来帮您彻底解决！"

**使用方式：**

##### 方式1：全局配置（.env文件）
```env
DEFAULT_PERSONA=cute  # 默认使用可爱热情型
```

##### 方式2：API请求时指定
```python
response = await client.send_message(
    message="我要退款",
    session_id="user_001",
    persona="cute"  # 这次使用可爱型
)
```

##### 方式3：会话级别设置
```python
# 为会话设置Persona（24小时有效）
await client.set_persona(
    session_id="user_001",
    persona="mature"
)

# 后续该会话的所有回复都会使用成熟冷静型
```

#### Persona API

**获取所有Persona：**
```bash
curl http://localhost:8000/api/v1/chat/personas
```

**设置会话Persona：**
```bash
curl -X POST http://localhost:8000/api/v1/chat/set-persona \
  -H "Content-Type: application/json" \
  -d '{"session_id": "user_001", "persona": "cute"}'
```

**获取会话Persona：**
```bash
curl http://localhost:8000/api/v1/chat/persona/user_001
```

#### 效果对比

**同一个查询"怎么退款？"的不同Persona回复：**

| Persona | 回复 |
|---------|------|
| friendly | "您好！我理解您需要退款。让我来帮您处理，请放心 😊" |
| cute | "好哒~ 退款问题我来帮您解决！别担心，很快搞定 ✨" |
| mature | "根据您的情况，退款流程如下：1. 申请 2. 审核 3. 到账" |
| professional | "针对您的退款需求，请提供订单号，我将为您查询进度" |
| humorous | "退款小case啦~ 我来施展魔法，帮您快速处理 💡" |
